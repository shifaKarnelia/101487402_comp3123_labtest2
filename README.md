Weather App built with React for COMP3123 Lab Test 2.

## Features
- Search weather by city
- Displays temperature, description, and icon
- Uses OpenWeatherMap API
- Hosted on Vercel

## API Used that i useed 
OpenWeatherMap Current Weather API  
https://openweathermap.org/

## Setup
1. Clone repo
2. Run `npm install`
3. Add your API key in App.jsx
4. Run `npm start`
